Memory — 长期记忆库

点击导航栏 Memory 进入。借鉴 Ombre Brain 理念的长期情感记忆系统。

核心理念：每条记忆带有情感坐标（效价 Valence / 唤醒度 Arousal）、重要性评分、自然衰减机制。不活跃的记忆权重随时间降低，高情绪强度的记忆衰减更慢——就像人的真实记忆。

星图：顶部星图以二维情感坐标展示所有记忆。横轴标注 效价 VALENCE（消极 → 积极），纵轴标注 唤醒 AROUSAL（平静 → 炽烈）。每颗星的颜色对应领域（情感/日常/创作/思考），大小反映重要性，明暗反映活跃度，金色标识被置顶的记忆。

仪表盘：页面顶部显示总计/置顶/未解决/已解决的统计卡片、领域分布条形图、Token 预算估算进度条（预算上限可在 API 页面「记忆系统设置」中调整）。

时间轴：左侧时间轴以行星形态展示记忆分布。每颗行星的颜色与星图一致、根据记忆所属领域着色，大小反映重要性。月份切换以菱形标记分隔。同一日期仅标注一次，避免视觉重复。点击行星可快速定位到对应记忆卡片。

记忆卡片：主页面显示标题、概括和创建者。卡片右下角的折叠指示器提示可点击展开，查看完整内容、标签和操作按钮。点击「编辑」在卡片内直接展开编辑面板。

置顶：最多可置顶 7 条记忆。置顶记忆会排列在列表顶部，卡片右上角显示钉子图标，与其他记忆之间有分隔线。置顶记忆权重恒定不衰减。

创建者标记：每条记忆标注由谁编写。手动创建显示你的昵称，AI 生成则显示 AI 昵称。用户编辑过 AI 生成的记忆后显示为联合编写。

创建记忆：手动点击「+ 新记忆」，或让 AI 自动生成。Chat 工具栏「写入记忆库」按钮、Blog / 密码日记本阅读页的「生成记忆」按钮、Letters 信件旁的「生成记忆」按钮均可触发。

写一句话：展开记忆卡片后，AI 编写的记忆会额外显示「写一句话」按钮。点击后，生成该记忆的 AI 会基于记忆内容、你们的关系设定和 TA 的人设，为你写下一句话。句话以引言格式展示在卡片内。

可见性控制：每条记忆可设为四种可见性——全部公开、仅指定可见、排除指定、完全私密。

Token 预算配置：在 API 页面「记忆系统设置」中可调整每次注入的字符上限（1000-6000）和每条记忆内容截取长度（100-800 字符）。调高预算让 AI 读取更多记忆上下文，但会增加 Token 消耗。

群聊记忆：创建群聊时可选择是否启用记忆。启用后群内 API 仅可读取全部公开的记忆。

导入/导出：记忆库支持独立导出和导入 JSON 文件，方便跨设备迁移。也包含在全站 Export 中。

API 配置教程

IB 的所有 AI 功能（Chat、Tea、Tarot、Story、Letters、Blog 评论）都需要一个有效的 AI API 密钥才能使用。这份教程会手把手教你：什么是 API、怎么获取、怎么填进 IB。

不管你是用官方的还是中转站的，填法基本一样——看完就会。

第零步 — 什么是 API？

你可以把 API 理解成"远程电话号码"。IB 本身不自带 AI，它需要通过这个号码打电话给云端的 AI 模型（比如 Claude、GPT），AI 听完你的话后回复，IB 再把回复展示给你。

这个电话号码就是 API Key（一串密钥字符），配合一个接口地址（Endpoint，告诉 IB 该往哪里打电话）和模型名称（告诉对方你要找谁接听），就能连通了。

API 密钥是按使用量计费的（不是月费），你发一条消息、AI 回一条消息，各消耗一点额度。一般日常聊天，一个月花不了几块钱。

方式一 — 使用官方 API（推荐新手）

直接从 AI 服务商官网注册账号，获取密钥。以最常用的两家为例：

Anthropic（Claude）官方 API

① 访问 console.anthropic.com，注册或登录你的 Anthropic 账号。

② 进入后台，找到 API Keys 页面，点击 Create Key，复制生成的密钥（以 sk-ant- 开头）。

③ 首次使用需在 Billing 页面绑定信用卡（Visa / Mastercard）并充值（最低 $5）。

④ 回到 IB → 点击导航栏 API → 点击 「+ 添加API」。

⑤ 按如下填写：

· 服务商：选 Claude (Anthropic)

· API Key：粘贴你刚复制的 sk-ant-… 密钥

· 模型：会自动填入 claude-sonnet-4-20250514（你也可以改成其他模型）

· 接口地址：会自动填入 https://api.anthropic.com/v1/messages（不用改）

⑥ 给 TA 起个昵称（这个名字会显示在好友列表里），点击保存，完成。

OpenAI（GPT）官方 API

① 访问 platform.openai.com，注册或登录。

② 进入 API Keys 页面，创建一个新密钥（以 sk- 开头），复制保存。

③ 在 Billing 中充值（最低 $5）。

④ 回到 IB → API → 「+ 添加API」。

⑤ 按如下填写：

· 服务商：选 GPT (OpenAI)

· API Key：粘贴你的 sk-… 密钥

· 模型：自动填入 gpt-4o-mini（可改为 gpt-4o 等）

· 接口地址：自动填入官方地址（不用改）

⑥ 起昵称，保存，完成。

DeepSeek / Gemini 同理

DeepSeek：去 platform.deepseek.com 注册获取密钥，IB 中服务商选 DeepSeek。

Gemini：去 aistudio.google.com 获取密钥，IB 中服务商选 Gemini (Google)。

选好服务商后，接口地址和默认模型都会自动填好，你只需要粘贴密钥就行。

方式二 — 使用中转站 API（国内用户推荐）

如果你无法直接访问国外官方 API（需要海外信用卡、网络不通等），可以使用中转站（也叫 API 代理 / 第三方转发）。中转站替你转发请求到官方服务，你只需要在中转站充值、拿到它给你的密钥和地址即可。

在中转站注册并获取信息

① 在你使用的中转站平台注册账号并充值。

② 在中转站后台找到以下三样东西（通常在"令牌"或"API 密钥"页面）：

· API Key：中转站给你的密钥（格式因平台而异，比如 sk-… 或其他格式）

· 接口地址（Endpoint / Base URL）：通常长这样 → https://你的中转站域名/v1/chat/completions

· 模型名称：你想用的模型，比如 claude-sonnet-4-20250514 或 gpt-4o（中转站页面一般会列出可用模型列表）

方法 A：选「自定义」（推荐，最通用）

绝大多数中转站都兼容 OpenAI 格式，这个方法通用性最好：

① IB → API → 「+ 添加API」

② 服务商：选 自定义

③ 接口地址：填你中转站给的完整地址，例如 https://api.example.com/v1/chat/completions

④ API Key：填中转站给你的密钥

⑤ 模型：填你要用的模型名（从中转站的模型列表中选一个，完整复制过来）

⑥ 起昵称，保存，完成。

这种方式使用 OpenAI 兼容协议（Bearer Token 认证），是市面上最通用的格式。

方法 B：选对应服务商，改接口地址

如果你的中转站明确支持 Anthropic 原生格式（x-api-key 认证），也可以这样：

① 服务商选 Claude (Anthropic)

② 把接口地址从官方的 https://api.anthropic.com/v1/messages 改成中转站的地址，例如 https://api.example.com/v1/messages

③ 填入中转站密钥和模型名

④ 保存即可

注意：这种方式会使用 Anthropic 原生的请求格式和认证方式。如果中转站不兼容，请用方法 A。

常见问题

填完之后怎么知道成功了？

保存后回到 Chat 页面（或点击右下角聊天气泡），你添加的 AI 会出现在好友列表中。给 TA 发一条消息试试——如果能收到回复，就说明配置成功了。

提示「API Key 无效」怎么办？

检查你的密钥是否复制完整（前后没有多余空格），以及中转站或官方账户是否还有余额。

提示网络错误 / 无法连接？

检查接口地址是否正确（特别是 https:// 开头不要漏掉），以及你的网络是否能访问该地址。使用官方 API 可能需要科学上网。

可以同时添加多个 API 吗？

可以，IB 最多支持 10 个。每个 API 会以独立的好友身份出现——你可以同时和 Claude 聊天、让 GPT 写信、用 DeepSeek 占卜。也支持把多个 AI 拉进一个群组里群聊。

密钥安全吗？

IB 是纯前端应用，你的密钥只存在你自己的浏览器里（IndexedDB），不会上传到任何服务器。调用 API 时，请求直接从你的浏览器发往 AI 服务（或中转站），IB 不经手。

官方 API 和中转站的区别？

官方 API 直连服务商，速度快、稳定，但可能需要海外信用卡和能访问海外网络。中转站是国内第三方代理，支持国内支付方式（支付宝/微信），无需科学上网，但速度和稳定性取决于中转站本身的质量。两者在 IB 中的功能完全一致，没有任何区别。
